# Design Context

## Project

- Project name: K-harness Ouroboros
- Collection date: 2026-05-08
- Output purpose: Korean YouTube motion explainer
- Format: 1920x1080 horizontal
- Target viewer: AI coding workflow에 관심 있는 개발자와 빌더
- Viewing situation: 데스크톱 또는 모바일 전체화면에서 1분대 설명 영상을 빠르게 시청

## Visual System

- Color tokens: dark navy base, cyan for system/process, yellow for emphasis, green for pass, pink for risk
- Font choices: Noto Sans KR
- Visual mood: technical, high-contrast, kinetic, diagram-first
- Motion personality: title cascade, card stagger, meter compression, gate activation, loop draw
- Caption safe zone: bottom caption remains fixed; important scene content stays above roughly `y=820`

## Constraints

- Forbidden styles: generic SaaS dashboard, decorative-only gradient blobs, product-free filler icons, caption overlap
- Unknowns: no official logo or UI asset was used in this version
