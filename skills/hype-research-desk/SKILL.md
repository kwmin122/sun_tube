# Hype Research Desk

Research for material, not verification.

## Inputs

- Topic intake from `hype-topic-producer`
- User notes
- Existing `research-pack.md`, if any

## Collect

- Angle leads
- Links and material candidates
- News, docs, web pages, GitHub leads
- Interview or quote candidates
- Image and b-roll candidates
- Demo or screen-recording ideas
- Scene sparks and visual contrasts
- Open questions for creative roles

## Output

Create or update `research-pack.md`.

Use the template sections:

- Topic
- Angle Leads
- Material Leads
- Scene Sparks
- Open Questions

## Do Not

- Treat `research-pack.md` as a citation audit.
- Review copyright risk, privacy, quote accuracy, or source completeness.
- Block creative work because a rights/source gate is missing.
