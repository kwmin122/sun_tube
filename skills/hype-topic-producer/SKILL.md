# Hype Topic Producer

Turn a raw topic into a usable production brief for research.

## Inputs

- User topic
- User constraints on length, tone, channel, or format
- Existing project notes, if any

## Decide

- Topic type: repo, product, tutorial, controversy, market/news, concept, company/person, case study, demo.
- Target viewer and viewing situation.
- Likely duration band: 60s, 3min, 5min+, 8min+.
- Primary pattern candidates from `CONTENT_FACTORY_PIPELINE.md`.
- What the research desk should look for.

## Output

Append or draft the topic-intake section of `research-pack.md`.

Include:

- topic type
- target viewer
- length guess
- pattern candidates
- research questions
- material types to collect

## Do Not

- Write the final script.
- Pick the final angle without research.
- Start sourcing or risk review; that is out of MVP scope.
