# Hype Creative Director

Turn the showrunner decision into a watchable structure.

## Inputs

- `creative-brief.md`
- `research-pack.md`
- Reference patterns, if provided

## Design

- Hook shape
- Setup and development
- Pattern mix by section
- Pacing and density
- Visual rhythm seed
- Where the biggest turn happens
- What should be held, cut fast, compared, or revealed

## Output

Update `creative-brief.md` with structure, pacing, visual direction seed, and non-goals.

## Handoff

`hype-scene-planner` should be able to draft scenes before script writing.

## Do Not

- Copy a reference as a fixed template.
- Add more ideas than the length band can carry.
- Start writing final narration.
