# hyperframes/scene-density

Use when a Hyperframes scene risks looking like an empty PPT card.

## Required For Rich Scenes

A rich explainer scene needs at least two of these:

- Korean information rows
- route tokens or packets
- before/after state
- progress rail or status ladder
- loop/gate/merge structure
- useful capture crop plus interpretation
- count, tick, or score state

## Fail If

- A large glass panel contains only a title and one short sentence.
- Cards appear without showing a mechanism.
- The same card grid layout repeats across three scenes.
- Motion is only fade + y + stagger.

## Better Patterns

- Show the verb: split, merge, score, revise, trigger, remember, compare.
- Use a source stamp plus a diagram instead of a full-page unreadable screenshot.
- Use lanes and packets instead of decorative connector lines.
